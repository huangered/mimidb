# MIMI Database

MIMI is a simple db for study

  - Refer to postgresql
  - Use cmake to cross-platform
  - Magic

## Folder structure
[access]
[catalog]
[commands]
[common]
[executor]
[parser]
[port]
[storage]
[util]

## Installation

MIMI db requires [CMAKE, GCC] to run.

### Build

```sh
$ mkdir build
$ cd build
$ cmake ..
$ cmake --build .
```
